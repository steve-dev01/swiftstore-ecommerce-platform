# swiftstore-ecommerce-platform

[![Backend CI](https://github.com/steve-dev01/swiftstore-ecommerce-platform/actions/workflows/backend-ci.yml/badge.svg?branch=main)](https://github.com/steve-dev01/swiftstore-ecommerce-platform/actions/workflows/backend-ci.yml)

🚀📈 A full-stack eCommerce platform that supports user shopping, secure payments, and role-based management for admins and staff. Built with React, Express, and PostgreSQL, it provides a streamlined shopping experience with integrated Stripe checkout.
